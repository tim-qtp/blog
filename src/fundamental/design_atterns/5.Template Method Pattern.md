简单来说，就是 **“固定流程，灵活实现”**。**定义一个固定的流程**（模板），这个流程由多个步骤组成。

### 以 JdbcTemplate 为例

JdbcTemplate 是 Spring 用来操作数据库的工具。它的流程是固定的：

1. 获取数据库连接。
2. 创建 SQL 语句。
3. 执行 SQL 语句。
4. 处理结果。
5. 关闭连接。

其中：

- **获取连接** 和 **关闭连接** 是固定的，Spring 已经帮你做好了。
- **创建 SQL 语句** 和 **处理结果** 是可变的，你可以根据自己的需求去实现。
